1.Windows 激活方法

1.打开 命令提示符(管理员)
2.执行以下命令

slmgr /skms kms.v0v.bid && slmgr /ato

如果激活失败使用命令提示符(管理员)执行以下命令更换密钥后重新激活（密钥在下文）
slmgr /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

查看系统版本（备用）
wmic os get caption
查看激活详情（备用）
slmgr /dlv
卸载密钥（备用）
slmgr /upk

2.Office 激活方法

1.Cmd进入office安装目录
打开命令提示符(管理员)执行以下命令(若Office安装在其他盘符，请自行将C改为对应盘符)

cd "C:\Program Files\Microsoft Office\Office14" 【备注：替换为你的 Office2010 安装目录】
cd "C:\Program Files\Microsoft Office\Office15" 【备注：替换为你的 Office2013 安装目录】
cd "C:\Program Files\Microsoft Office\Office16" 【备注：替换为你的 Office2016 安装目录】

如果你不知道你的office装在哪个目录，可以打开一个程序比如word，然后用打开任务管理器右键选择“打开文件所在的位置”

2.执行命令激活Office软件

cscript ospp.vbs /sethst:kms.v0v.bid && cscript ospp.vbs /act

查询Office激活详情（备用）
cscript ospp.vbs /dstatus

3.激活说明

KMS批量激活密钥广泛用于机房服务器操作系统和大型企业内部批量激活员工电脑，限企业内网使用。目前公开的KMS服务器封杀力度很大，为了节省大家的时间与精力，请低调使用，转载请注明：https://v0v.bid 。

KMS 激活有 180 天期限，此期限称为激活有效间隔。若要保持激活状态，您的系统必须通过至少每 180 天连接一次 KMS 服务器来续订激活。
默认情况下，系统每 7 天自动进行一次激活续订尝试。在续订客户端激活之后，激活有效间隔重新开始。

综上所述，只要您的电脑不超过 180 天以上无法连接互联网，就无需进行任何操作，系统会自行续期保持激活状态。即永久激活。

如若你无法保证半年内上一次网，或者你有心理上的强迫症。那么下拉到本文末，按提示将你的电脑升级为中国政府版。
你只需要保证 400年 连接一次互联网，即可永久使用正版的win10系统。

6.激活失败如何排错

1.你的系统/OFFICE是否是批量VL版本
2.是否以管理员权限运行CMD命令提示符
win10:点开开始菜单，在搜索框中输入“cmd”，在搜索结果中，对着命令提示符程序，单击鼠标右键，菜单中点击选择“以管理员身份运行”
win7:点开开始菜单，再点击所有应用（程序），在Windows系统（Win7时附件中），找到并右键单击，菜单中选择“以管理员身份运行”
3.你的系统/OFFICE是否修改过KEY/未安装GVLK KEY
4.检查你的网络连接
5.本地的解析不对,或网络问题
